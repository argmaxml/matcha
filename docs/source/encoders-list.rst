Encoders
================

Recsplain comes with a variety of encoders. 

.. note:: 
   The code for the encoders is in `encoders.py <https://github.com/argmaxml/recsplain/blob/master/recsplain/encoders.py>`_

Here is the list.

NumericEncoder

OneHotEncoder

StrictOneHotEncoder

OrdinalEncoder

BinEncoder

BinOrdinalEncoder

HierarchyEncoder

NumpyEncoder

JSONEncoder

QwakEncoder
